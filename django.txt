start een project in root folder:
django-admin startproject <naam project>
cd <naam project>
python manage.py migrate
python manage.py startapp <appnaam>
na het aanmaken van je model in models.py
python manage.py makemigrations <appnnaam>
daarna voer je de aangemaakte migration uit
python manage.py migrate (push deze statements vanuit de migration naar de database)
winpty python manage.py createsuperuser
om een admin aan te maken
python manage.py runserver om de server op te starten
navigeer naar /admin/ en log in
